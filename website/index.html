What's Jua?

Jua is a day-long gender focused coding social event where participants team up to code projects, learn new skills, and have fun! Participants don't need to have any prior coding experience; we'll help them create magic! They might build their first ever website or video game <3 We invite girls to join us for 7 hours of creating, exploring and sharing!

Frequently Asked Questions

Who can attend?

This event is for all teenagers aged 13-17 who are interested in coding!

What if I have never coded before?

Of course, this event is beginner-friendly and we'll have workshops and mentors to help you!

How much does it cost to Attend?

Nothing; the event is free!

What should I bring?

Just your laptop and yourself! But if you can't bring a laptop please let us know early so we can get you one.

Where is the event located?

Nairobi Game Development Centre, Diamond Plaza, 6th floor, Nairobi.
